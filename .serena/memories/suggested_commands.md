# Suggested Commands

- `npm run start` starts Expo with Storybook disabled.
- `npm run start:iphone` starts the LAN iPhone workflow via `scripts/start-expo-lan.js`.
- `npm run start:dev-client` starts Expo for a dev client.
- `npm run storybook:mobile` starts mobile Storybook on port 8084.
- `npm run storybook:web` starts Storybook web preview.
- `npm run typecheck` runs `tsc --noEmit`.
- `npm run design:audit` runs the design token audit.
- Darwin shell note: quote route paths containing brackets, e.g. `'app/workouts/[workoutId]/session.tsx'`, to avoid zsh glob errors.
