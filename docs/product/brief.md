# Product Brief

## Product

Mobile app for personal trainers working with clients in a gym environment.

## Primary User

Personal trainer who needs fast access to today's schedule, client context, workout plan, previous exercise results, and set logging.

## First Phase Scope

- No backend.
- Mock data only.
- Expo React Native app.
- Reusable design-system components.
- Dev-kit screen for component verification.
- Figma component library aligned to code.

## Core Jobs

- See today's workouts.
- Open client profile.
- Conduct a workout session.
- Add and complete sets quickly.
- See previous result for an exercise.
- Create or adjust a workout.

## UX Priorities

- Large touch targets.
- Fast weight/reps input.
- Clear completed/error states.
- Minimal cognitive load during training.
- Consistent tokenized UI.
