# MVP Scope

## In Scope

- Today screen.
- Clients list.
- Client profile.
- Workout session.
- Workout builder.
- Local mock data.
- Theme tokens generated from Figma design tokens.
- Dev-kit route for component QA.

## Out Of Scope

- Backend.
- Auth.
- Payments.
- Push notifications.
- Real calendar sync.
- PDF generation pipeline beyond preview components.

## Acceptance Criteria

- Components are reusable and tokenized.
- Screens use canonical components.
- TypeScript passes.
- Design audit passes once the theme is migrated to the current token shape.
