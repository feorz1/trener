# Typography

## Metadata

- Tier: foundations
- Source: `DESIGN.md`
- Figma styles: `Typography/*`
- Code: `theme.typography`

## Groups

- `Typography/Display/*` -> `theme.typography.display.*`
- `Typography/Body/*` -> `theme.typography.body.*`
- `Typography/Caption` -> `theme.typography.caption`
- `Typography/Button/MD` -> `theme.typography.button.md`

## Rules

- Every Figma text node in canonical components must use a text style.
- Every React Native component must use `theme.typography.*`.
- Do not define component-local font sizes, weights, or line heights.
