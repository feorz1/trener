# Elevation

## Metadata

- Tier: foundations
- Source: `DESIGN.md`
- Code: `theme.shadows`

## Rules

- Wise-style elevation relies primarily on surface contrast.
- Use shadows sparingly.
- Component code must use `theme.shadows.*` when elevation is required.
- Raw shadow objects should not appear inside component files unless a missing token is being added to `src/theme/shadows.ts`.
