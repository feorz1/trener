# Mobile Screen Layout

## Metadata

- Tier: patterns
- Viewport: `375 x 812`

## Overview

Trainer app screens should optimize for fast use in a gym: visible schedule context, large controls, and minimal nested navigation.

## Rules

- Use canonical components from `component-registry.json`.
- Use `theme.spacing.*` for section gaps.
- Keep primary actions near the current task context.
- Do not build page sections from one-off card layers when a canonical component exists.
